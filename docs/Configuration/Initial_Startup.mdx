## User Interface

Once Preroll Plus is running, you can view the Preroll Plus user interface by navigating to:

```
http://<ip_address_of_server>:<port>
```

:::info
The IP address will be the address where Preroll Plus is running. The port number will depend on how the application was installed and started up. (See [Installation](/docs/Installation))

Default Port: `4949`
:::

## Plex Login

When the application starts up for the first time a new settings file will be created and you will be prompted to sign in with your Plex account.

:::info
The reason for logging into your Plex account is to retrieve information about the location of your Plex server as well as obtain a unique Plex token for Preroll Plus to use while accessing your Plex Server.
:::

:::warning Don't forget!
After logging in with your Plex account you will automatically be routed to the [Settings](/docs/Configuration/Settings) page for initial configuration, but don't forget to configure your [Webhook](/docs/Configuration/Webhook)!
:::
