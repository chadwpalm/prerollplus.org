title: Calendar

sidebar_position: 2
---

The calendar page lets you see which sequences will play on a particular month in a calendar view. Click the arrows on the top left of the page to scroll through months. Click `today` to jump to the current day.

:::tip
If you hover over a sequence name it will show the list of buckets used in that sequence.
:::

:::note
The calendar is restricted to only viewing the current year and the following year.
:::
